Math And Semantics

Keep

  • `input`: one primary user-message fact.
  • `combo`: one folded middle summary.
  • `tail`: only the explicit stations worth reading.
  • `pending` / `error`: state, not extra structure.

Omit

  • Repeated metrics already implied by the header or line.
  • Single tool counts and single-part evidence.
  • Full payloads that belong in detail.
  • Any chip that only restates a visible sentence.
