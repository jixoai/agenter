Rules

  • Neutral tones own layout, chrome, borders, and rails.
  • Semantic hues belong to message-part role and status only.
  • Semantic hues use OKLCH with a shared lightness target, so hue changes do not accidentally create hierarchy.
  • Chips derive ordered color stops from their semantic tokens; compressed chips keep meaning through the gradient even when labels disappear.
  • `Combo` renders the folded part sequence as multiple stops; it is structure, not a new factual part kind.
  • `Pending` stays warm like tool work, but uses gold instead of orange so unfinished latest work is not confused with a tool call.
  • `Error` is a terminal failed fact, so it stays stable and red rather than sharing the running animation.
